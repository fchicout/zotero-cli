# Command: `system`

Diagnostics and environment information for the Zotero CLI.

## Verbs

### `info`
Display current configuration paths, library IDs, and authentication status.

**Usage:**
```bash
zotero-cli system info
```

---

### `groups`
List all Zotero Groups the authenticated user has access to, including their internal IDs.

**Usage:**
```bash
zotero-cli system groups
```
