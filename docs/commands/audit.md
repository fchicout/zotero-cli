# Command: `audit` (Deprecated)

This command is deprecated and has been consolidated into the **[`slr audit`](slr.md#audit)** command.

Please use `zotero-cli slr audit` for all auditing workflows.

## Consolidated Verbs

The following verbs have been moved to the `slr` command:
* `check` (moved to `slr audit`)
* `import-csv` (moved to `slr import-csv`)
