# Command: `find`

Discovery and search tools for expanding your library.

## Verbs

### `arxiv`
Search the ArXiv repository using advanced query syntax.

**Usage:**
```bash
zotero-cli find arxiv --query "ti:Transformer AND cat:cs.LG"
```

**Parameters:**
*   `--query`: The ArXiv query string.
*   `--file`: Optional path to a file containing the query.
*   `--verbose`: Show detailed search logs.
