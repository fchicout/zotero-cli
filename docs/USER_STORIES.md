# User Stories

## Research & Ingestion
- **US1:** As a researcher, I want to import papers directly from arXiv using complex queries so that I can automate the initial search phase of my SLR.
- **US2:** As a researcher, I want to import RIS and BibTeX files exported from other databases (like Scopus or WoS) so that I can centralize my references in Zotero.
- **US3:** As a researcher, I want the system to automatically find and download open-access PDFs for my items so that I don't have to search for them manually.

## Screening & Selection
- **US4:** As a reviewer, I want a high-velocity keyboard-driven interface for screening titles and abstracts so that I can process hundreds of papers per hour.
- **US5:** As a reviewer, I want to record the specific reason for excluding a paper so that my selection process is transparent and scientifically rigorous.
- **US6:** As a team lead, I want screening decisions to be stored as child notes in Zotero so that multiple researchers can see the audit trail without external databases.

## Reporting & Synthesis
- **US7:** As an author, I want to generate a PRISMA diagram with one command so that I can meet the reporting standards of high-impact journals.
- **US8:** As a researcher, I want to ask natural language questions about my library's content so that I can quickly find relationships between different studies.
- **US9:** As a student, I want to export my papers' metadata and full-text to Markdown so that I can use them in my personal knowledge management system (like Obsidian).

## Maintenance & Quality
- **US10:** As a librarian, I want to identify items missing abstracts or DOIs across a large collection so that I can ensure the quality of my bibliographic data.
- **US11:** As a researcher, I want to create an immutable snapshot of my library at a specific point in time so that I can guarantee the reproducibility of my results.
